Opdracht 4

Hieronder zie je een box-model Leg uit wat de volgende termen betekenen: content, padding, margin en border.

Content: Dit is wat er wordt vertoond op de scherm.

Padding: Dat is de opvulling van het element, dus de spatie binnen het element

Margin: Dit is de ruimte aan de buitenkant vanuit het element

Border: Dit is het rand van het element
