Kun je enkele voorbeelden geven van pseudo-elementen ( minimaal 4 )? Beschrijf met je eigen woorden wat deze pseudo-elementen inhouden.

Er zijn 14 verschillende pseudo elementen. Je hebt in de voorgaande opdracht reeds 4 kunnen benoemen. Kun je de andere 10 ook benoemen en aangeven wat ze precies doen?

::after

Hier wordt na het geselecteerd element een nieuw element aangemaakt

                                .after::after{
                                    content: "Ik ben toegevoegd door CSS pseudo element \"After\" ";
                                }

::backdrop

Deze code wordt uitgevoerd op het moment dat er een element in fullscreen mode word geplaatst en die is even groot als de viewport

                                video::backdrop{
                                    background: rgba(86, 204, 219, 0.842);
                                }

::cue

Deze pseudo element wordt gebruikt om de ondertiteling in media met een VTT tracks te stylen

                                ::cue{
                                    text-decoration: italic;
                                    text-decoration: underline;
                                    background-color: hotpink;
                                }

Hierdoor wordt de ondertiteling schuingedrukt, onderlijnd en met een "hotpink" achtergrond

::cue-region (Experimental)

De ::cue-region gebruikt bestaande cues om een groepering te maken en deze te stylen. Overigens nog geen browser ondersteuning!

                                ::cue-region{
                                    font-size: 4px;
                                    color: red;
                                }

De ondertiteling krijgt een rode letterkleur en is 4px groot

::before

Hier wordt vóór het geselecteerd element een nieuw element aangemaakt

                                .before::before{
                                    content: "Ik ben een product van het ::before pseudo element";
                                    text-decoration: underline;
                                    background-color: honeydew;
                                }

::placeholder

Styling geven aan de placeholder attribuut van bijvoorbeeld een textarea-element

                                .placeholder textarea::placeholder{
                                    color: dodgerblue;
                                    background-color: rgba(226, 91, 37, 0.349);
                                }

::part()

Dit element wordt gebruikt om styling te geven aan een custom element die in de shadow DOM zit en dus is niet met normale methodiek te stylen. De part-attribuut is nodig om de custom-element te kunnen benaderen

                                    custon_element::part(waarde_van_part-attr){
                                        background-color: red;
                                        color: white;
                                    }

::marker

Hiermee is het markeer gedeelte van een lijst-item (li) te stylen

                                .marker ul li::marker{
                                    font-weight: bold;
                                    color: forestgreen;
                                }

                                .marker ol li::marker{
                                    font-size: 20px;
                                    font-style: italic;
                                }

::slotted()

Hiermee kan de slots elementen binnen een template-element geselecteerd worden voor styling

                                HTML:
                                <template id="person-template">
                                    <div>
                                      <h2>Personal ID Card</h2>
                                      <slot name="person-name">NAME MISSING</slot>
                                      <ul>
                                        <li><slot name="person-age">AGE MISSING</slot></li>
                                        <li><slot name="person-occupation">OCCUPATION MISSING</slot></li>
                                      </ul>
                                    </div>
                                  </template>

                                  CSS:
                                  ::slotted(person-name){
                                      font-size: 20px;
                                  }

::spelling-error() - (Experimental)

Op het moment dat er een spelfout geconstateerd wordt door de browser wordt er visueel duidelijk gemaakt dat er een fout(en) zijn gevonden. De styling van dit visueel kenbaar maken kan door spelling-error gestyled

                                <p spellcheck="true">Guiter</p>

Er is geen browser die dit pseudo element ondersteund!

::selection()

Hiermee kan een styling gegeven worden aan een geselecteerd tekst

                                .selection p::selection{
                                    background-color: violet;
                                    color: turquoise;
                                }
