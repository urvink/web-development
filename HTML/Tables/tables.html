Tables - HTML (Week1)

10.Om een tabel te maken heb je volgende elementen nodig:
  • <table>
  • <thead>
  • <tbody>
  • <tfoot>
  • <th>
  • <tr>
  • <td>
W3Schools Table Tag
<table></table>
De tabel tag: HTML tabel. Tabel element met één of meer tr, th, td elements.
<thead></thead>
De thead tag: groepering van headings, bijvoorbeeld Maand, Salaris.
<tbody></tbody>
De tbody tag: groepering van body content in HTML tabel, bijvoorbeeld Januari, $100.
<th></th>
De th tag: 'header cell', bevat heading informatie, bijvoorbeeld Maand.
<tr></tr>
De tr tag: Rij in HTML table.
<td></td>
De td tag: standaard cell, bijvoorbeeld Januari
