1. Wat is het verschil tussen het internet en het world wide web?

World Wide Web, is een stukje van het internet. Het internet is een wereldwijde netwerk die miljoenen computers met elkaar verbindt, denk aan hardware, routers, kabels, computers enz. De WWW is een grote verzameling van pagina's van informatie die je met een simpele muisklik kan bekijken.

Een attribuut geeft extra informatie over een element. Attributen worden altijd geschreven in de start-tag en worden gevolgd door een gelijk teken. 

                    - Style attribute 
                        <h2 style="background-color: #ff0000;">Mijn eerste HTML pagina.</h2>

7. Wat is het verschil tussen inline en block elementen?

Een block-level element begint altijd op een nieuwe regel en neemt de volle breedte van het browservenster in. Het strekt zich uit naar links en rechts uit zo ver als het kan. Een inline element start niet op een nieuwe regel en beslaat slechts zo veel breedte als het nodig heeft om wat erin staat weer te geven https://www.w3schools.com/html/html_blocks.asp

8. Wat betekent nesting? Codeer een voorbeeld met nesting erin.

Nesting betekent elementen die binnen elementen worden geplaatst It is a common practice to place multiple <div> inside another <div>. This is usually referred to as "nesting" elements and allows for further dividing elements into subsections or aid developers with CSS styling.

The <div class="outer-div"> is used to group together two <div class="inner-div"> elements; each containing a <p> element. 

                    <div class="outer-div">
                        <div class="inner-div">
                                <p>This is a paragraph</p>
                        </div>
                        <div class="inner-div">
                            <p>This is another paragraph</p>
                        </div>
                    </div>
