1. Wat is het verschil tussen het internet en het world wide web? 
World Wide Web, is een stukje van het internet. Het internet is een wereldwijde netwerk die miljoenen computers met elkaar verbindt, denk aan hardware, routers, kabels, computers enz. De WWW is een grote verzameling van pagina's van informatie die je met een simpele muisklik kan bekijken.

2. Wat is het verschil tussen HTML, CSS en JavaScript? 
HTML geeft structuur aan de website
CSS geeft opmaak aan de website 
Javascript gaat om de functionaliteit van de website (interactief maken van de website)
