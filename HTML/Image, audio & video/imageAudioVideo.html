Image, Audio & Video - HTML (Week1)

Het plaatsen van foto's op een website is een belangrijke toevoeging voor velen. Zo bestaan de meeste websites uit een grote collectie van foto's. Vaak kun je logo's, teamfoto's of andere soorten afbeeldingen terugvinden op een website. Het stylen van de afbeeldingen komt later in het onderdeel CSS aanbod.

1. Bij het inladen van een bestand is het noodzakelijk om de locatie van het bestand aan te geven. Dit kan op 2 manieren: ● Absolute path ● Relative path Wat houden deze 2 paths in?

  • Absolute pathe: Een complete referentie naar een resource vb.: https://www.google.com/fotos/dsc01205.jpg
  • Relative path: Een referentie naar een resource vanuit de huidige locqatie vb.: ./images/dsc23405.jpg

Een image element is voor het inladen van een afbeelding. Afbeeldingen kunnen van verschillende formaat zijn. Bijvoorbeeld, een afbeelding kan een JPG formaat zijn.

2. Zoek uit welke formaat afbeeldingen er nog meer kan zijn

  • *.jpg voor jpeg-bestanden - jpeg images
  • *.png voor png-bestanden - png images (lossless afbeeldingsformaat met een alpha-laag)
  • *.bmp voor bmp-bestanden - Bitmap images
  • *.svg voor svg-bestanden - Vector graphics

Een afbeelding tonen is afhankelijk van of de browser het ondersteunt

3.Zoek uit welke browsers er zijn en maak een lijst van je antwoord

  • Google Chrome Opera browser Safari browser Firefox browser

4. Onderzoek hoe het plaatsen van een foto gedaan wordt en plaats deze boven en onderaan je website door zowel gebruik te maken van de absolute path en relative path.

5. Wat houdt de alt attribute in? En waarom is het handig om de alt attribuut te gebruiken?

ALT-attribuut gebruik je bij een img-tag om op het moment dat de afbeelding niet werkt (gevonden kan worden) wordt de tekst die er staat bij de alt-attribuut weergegeven.

Een audio bestand heeft verschillende formaten. Niet alle formaten worden ondersteund door de browser. Hiervoor is een oplossing voor.

Het voordeel van het gebruik van de source element, betekent dat je meerdere bronnen kan toevoegen aan de AUDIO-element.

Door height en/of width-attribuute te gebruiken kan je de hoogte en/ of breedte van de video bepalen
